# OpenGarageFlow

OpenGarageFlow is an open-source platform for managing automotive workshops.

The project aims to provide a modern, self-hostable, and SaaS-ready solution for managing customers, vehicles, inventory, work orders, employees, and day-to-day workshop operations.

> **Project Status:** Early Development

## Features

The initial release will focus on:

* Authentication
* Customer Management
* Vehicle Management
* Work Orders
* Inventory
* Dashboard

Additional modules will be introduced in future releases.
